There are many currencies in the game, and most of them can be interchanged.

# Cards

Acquired by:
- World drops
- Match reward
- Draft wins
- Shop purchases
- Quests

Spent on:
- Selling to shops

# Gold

Acquired by:
- World drop
- Match reward
- Draft reward
- Quests

Spent on:
- Cards and items from shops
- Drafts games
- Crafting cards
- Shards

# Shards

Acquired by
- World drop
- Quest reward

Spent on
- Crafting cards
- Shop Re-rolls
- Gold exchange

# Challenge Coins

Acquired by
- At the start of the game

Spent on
- Drafts
